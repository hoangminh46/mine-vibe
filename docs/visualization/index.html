Mine Workflow Overview

Catalog workflow hiện tại được quyết định bởi các file đang có trong workflows/coding/ và workflows/system/. Bộ workflow cốt lõi của Mine hiện tập trung vào discovery, implementation, validation và memory.

Discovery & Requirements

Làm rõ user, problem, scope, requirements và implementation plan.

  • /brainstorm
  • /requirements
  • /plan

Design & Implementation

Chốt visual direction, triển khai code, mock API và refactor an toàn.

  • /visualize
  • /code
  • /mock-api
  • /refactor

Validation & Recovery

Đo mức confidence hiện tại, debug khi cần, audit theo scope và chọn bước tiếp theo.

  • /test
  • /debug
  • /audit
  • /next

Memory & Utilities

Lưu context theo Git, recap phiên mới và hỗ trợ cá nhân hóa/cập nhật Mine.

  • /save-brain
  • /recap
  • /customize
  • /mine-update

Flow khuyến nghị

/brainstorm -> /requirements -> /plan -> /visualize -> /code -> /test -> /save-brain
/plan -> /visualize -> /code -> /test -> /save-brain
/mock-api -> /code -> /test
/recap -> /next -> /code hoặc /debug hoặc /plan

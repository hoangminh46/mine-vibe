# Workflow Flowchart

## Nhóm workflow hiện tại

```text
Discovery & Planning
/brainstorm -> /requirements -> /plan -> /visualize

Build & Validate
/code -> /test -> /debug
              -> /refactor
              -> /audit
              -> /mock-api

Memory & Guidance
/save-brain <-> /recap
             -> /next
             -> /customize
             -> /mine-update
```

## Flow khuyến nghị

### Ý tưởng mới

```text
/brainstorm -> /requirements -> /plan -> /visualize -> /code -> /test -> /save-brain
```

### Quay lại làm việc

```text
/recap -> /code hoặc /debug -> /test -> /save-brain
```

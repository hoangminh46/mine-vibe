# Cover Page Template - Trường ĐH Giáo Dục

## Layout Structure

The cover page must follow this exact layout (top to bottom, centered):

```
[Line 1] ĐẠI HỌC QUỐC GIA HÀ NỘI
         Font: Times New Roman, 15.5pt, Regular, Center

[Line 2] TRƯỜNG ĐẠI HỌC GIÁO DỤC
         Font: Times New Roman, 15.5pt, Bold, Center

[Spacing: ~3 empty lines]

[Line 3] BÀI THI HẾT HỌC PHẦN / TIỂU LUẬN CUỐI KÌ
         Font: Times New Roman, 20pt, Bold, Center

[Line 4] {TÊN HỌC PHẦN - UPPERCASE}
         Font: Times New Roman, 18pt, Bold, Center

[Line 5] Mã học phần: {MÃ HỌC PHẦN}
         Font: Times New Roman, 14pt, Bold, Center

[Spacing: ~4 empty lines]

[Info block - Left aligned, 14pt, Bold]:
  Giảng viên: {TÊN GIẢNG VIÊN}
  Học viên: {TÊN HỌC VIÊN}
  Ngày sinh: {NGÀY SINH}
  Mã học viên: {MSHV}
  Khoá: {KHÓA}
  Ngành: {NGÀNH}

[Spacing: ~3 empty lines]

[Footer] Hà Nội – {NĂM}
         Font: Times New Roman, 14pt, Bold, Center
```

## Notes
- No page number on cover page
- All text centered except the info block (left-aligned)
- Info block indented approximately 4cm from left margin
- Year defaults to current year
- "BÀI THI HẾT HỌC PHẦN" or "TIỂU LUẬN CUỐI KÌ" depending on assignment type

# DOCX Format Specification - ĐH Giáo Dục

## Page Setup
| Property | Value | OOXML |
|---|---|---|
| Page size | A4 (210mm x 297mm) | w:w="11906" w:h="16838" |
| Left margin | 3cm | 1701 twips |
| Right margin | 2.5cm | 1418 twips |
| Top margin | 2cm | 1134 twips |
| Bottom margin | 2cm | 1134 twips |

## Typography

### Body Text
- Font: Times New Roman
- Size: 14pt (28 half-points)
- Line spacing: 1.5 (360 twips, auto)
- Alignment: Justify (both)
- Color: Black (#000000)

### Heading Level 1 (Section titles: 1., 2., 3....)
- Font: Times New Roman
- Size: 14pt  
- Bold: Yes
- Line spacing: 1.5
- Alignment: Left
- Space before: 12pt
- Space after: 6pt

### Heading Level 2 (Sub-sections: 1.1., 1.2.)
- Font: Times New Roman
- Size: 14pt
- Bold: Yes
- Line spacing: 1.5
- Alignment: Left
- Space before: 6pt
- Space after: 3pt

### Heading Level 3 (Sub-sub-sections: 1.1.1.)
- Font: Times New Roman
- Size: 14pt
- Bold + Italic: Yes
- Line spacing: 1.5
- Alignment: Left

## Table of Contents
- Title "MỤC LỤC" centered, bold, 14pt
- Each entry: section number + title + page number
- Dot leaders between title and page number
- Font: Times New Roman, 14pt

## References Section
- Title "TÀI LIỆU THAM KHẢO" centered, bold, 14pt
- Format per entry: [STT]. Tác giả (Năm). *Tên tác phẩm*. Nhà xuất bản, Trang.
- Sorted alphabetically by author last name
- Minimum 3 references
- Font: Times New Roman, 14pt

## Page Numbering
- No number on cover page
- Arabic numerals starting from page 2 (or page of TOC)
- Position: Bottom center
- Font: Times New Roman, 12pt
